# Data-Analysis-Practice-1-60-Excel-Cell-References
This repository contains my Excel practice on Relative, Absolute, and Mixed cell references, along with Auto Fill Series (1–20). The goal is to understand how formulas work and how cell references change when copied. This is part of my Data Analysis Practice series (1/60).

## Project Objective
The main objective of this project is to understand and practice different types of cell references in Microsoft Excel.
This practice helps to:
•	Learn how formulas work in Excel
•	Understand how cell references change when copied
•	Improve basic data analysis skills
•	Build a strong foundation for advanced Excel tasks

## Questions Covered
1.	Relative Reference
2.	Absolute Reference
3.	Mixed Reference (Row Locked)
4.	Mixed Reference (Column Locked)
5.	Auto Reference (1 to 20) using Fill and Series
   
## KPI (Key Performance Indicators)
•	Correct use of formulas
•	Correct use of relative reference (A1)
•	Correct use of absolute reference ($A$1)
•	Correct use of mixed references ($A1 and A$1)
•	Successfully generated numbers 1 to 20 using Fill and Series
•	No formula errors

## Process
1.	Created a simple dataset in Excel.
2.	Applied Relative Reference formula and copied it.
3.	Applied Absolute Reference using $ sign.
4.	Used Mixed Reference with row locked and column locked.
5.	Used Fill Handle to generate numbers 1 to 20.
6.	Checked all outputs for correctness.
